# CryTools
Collection of tools for various CRI formats

## Programs
 
  - **sfdmux**    - program for muxing mpeg1video and ADX audio.
  - **test_adx**  - prints info about an ADX file.
  - **test_mpeg** - prints info about an MPEG file.
